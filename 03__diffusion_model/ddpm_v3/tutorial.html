Training and Image Generation

This tutorial explains how to train the DDPM v3 model and generate images.

Training

  1. Create a configuration YAML based on configs/config_simple01.yaml.
  2. Run python run.py --config my_config.yaml --training.
  3. Training logs and checkpoints are written to the output directory specified in the configuration.

Image generation

  1. Update the IMAGE_GENERATION section in your configuration.
  2. Run python run.py --config my_config.yaml --imgen.
  3. Generated images will be saved under the directory specified by GEN_OUTPUT_DIR.
